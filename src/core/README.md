# Core Layer

## Ответственность

Слой `core` содержит **доменную логику приложения**, независимую от конкретных внешних API и сервисов.

## Содержимое

- **config.py** — управление конфигурацией приложения (чтение переменных окружения, пути к credential-файлам)
- Будущие модули: модели данных, бизнес-логика обработки видео, валидация и т.п.

## Правила зависимости

**ВАЖНО:** Слой `core` НЕ ДОЛЖЕН зависеть от:
- `adapters` — никаких импортов из адаптеров
- `cli` — никаких импортов из CLI-слоя
- Конкретных внешних API (YouTube, и т.п.)

Слой `core` может зависеть только от:
- Стандартной библиотеки Python
- Общих библиотек (pydantic, dataclasses и т.п.)
- Других модулей внутри `core`

Это обеспечивает **независимость доменной логики** и возможность её переиспользования.
