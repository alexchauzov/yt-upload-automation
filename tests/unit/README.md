# Unit Tests

## Назначение

Юнит-тесты проверяют **отдельные компоненты в изоляции** без обращения к внешним сервисам.

## Что тестируем

- Модули из `core` — доменная логика, конфигурация, валидация
- Отдельные функции из `adapters` с использованием моков
- Утилиты из `cli`

## Принципы

- Каждый тест должен быть **быстрым** (миллисекунды)
- Тесты **независимы** друг от друга
- Используем **моки и стабы** для внешних зависимостей
- Не обращаемся к реальным API, файловой системе, базам данных

## Структура

```
unit/
  test_config.py           # Tests for core.config
  test_youtube_client.py   # Tests for adapters.youtube (mocked)
  ...
```

## Запуск

```bash
pytest tests/unit/
```
