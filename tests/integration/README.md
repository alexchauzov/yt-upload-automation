# Integration Tests

## Назначение

Интеграционные тесты проверяют **взаимодействие компонентов** и работу с реальными (или близкими к реальным) внешними зависимостями.

## Что тестируем

- Интеграцию между `core`, `adapters` и `cli`
- Работу с YouTube API (опционально, через тестовый аккаунт или в режиме dry-run)
- End-to-end сценарии (загрузка видео, аутентификация и т.п.)

## Принципы

- Тесты могут быть **медленнее** юнит-тестов
- Могут требовать **настройки окружения** (credentials, тестовые данные)
- Должны быть **идемпотентными** — можно запускать многократно
- По возможности используем **тестовые аккаунты** или sandbox-режим API

## Структура

```
integration/
  test_youtube_upload.py      # Full upload flow test
  test_auth_flow.py            # OAuth flow test
  ...
```

## Запуск

```bash
pytest tests/integration/
```

Некоторые тесты могут требовать переменных окружения с тестовыми credentials.
